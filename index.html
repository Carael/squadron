A framework to simplify your integration tests

Squadron is a useful framework which enables you to write integration tests against services without any overhead. Mostly supported services are hosted in containers which is managed by the framework. Other services which are not supported to run in conainer are implemented as a proxy against original SaaS platform.

  1. Install Nuget Package:

Install the Squadron nuget package for MongoDB (or other supported service) within your test project:

dotnet add package Squadron.Mongo

  1. Use provided resource:

Inject the provided resource into your test class constructor and initialize your repository or service:

public class AccountRepositoryTests
 : IClassFixture<MongoResource>
{
    private readonly IAccountRepository _accountRepository;

    public AccountRepositoryTests(
        MongoResources mongoResources)
    {
        var database = mongoResources.CreateDatabase();
        _accountRepository = new AccountRepository(database);
    }
}

  1. Use repository/service in your test:

            [Fact]
            public void CreateAccount_AccountExists()
            {
                // arrange
                var account = new Account();
            
                // act
                var addedAccount = _accountRepository.Add(account);
            
                // assert
                Snapshot.Match(addedAccount);
            }
            ```
